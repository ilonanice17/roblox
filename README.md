# 📖 Random Quote Generator (Генератор случайных цитат)

## 👤 Автор
[Абдуллин Айнур Альмирович]

## 📝 Описание
Графическое приложение на Python с использованием библиотеки `tkinter`, предназначенное для генерации случайных цитат из встроенной базы данных. Программа поддерживает фильтрацию по автору и теме, ведёт историю сгенерированных цитат и обеспечивает сохранение/загрузку всех данных в формате JSON. Интерфейс полностью адаптивен и работает на любой ОС с поддержкой Python GUI.

## ✨ Функционал
- 🎲 **Генерация цитат**: случайный выбор из предустановленной базы с учётом активных фильтров
- 🔍 **Фильтрация**: выпадающие списки по автору и теме (регистрозависимое точное совпадение)
- 📜 **История**: отображение последних 20 сгенерированных цитат в прокручиваемом поле
- 💾 **Персистентность**: сохранение и загрузка цитат + истории в файл `quotes_data.json`
- 🧹 **Управление**: кнопка очистки истории и ручного сохранения данных
- 📦 **Без зависимостей**: работают только стандартные модули Python (`tkinter`, `json`, `os`, `random`)

## 🛠 Требования
- Python 3.8 или выше
- Операционная система с поддержкой GUI (Windows, macOS, Linux)
- Встроенная библиотека `tkinter` (обычно устанавливается вместе с Python)

## ▶️ Как запустить
1. Сохраните код в файл `main.py`
2. Откройте терминал в папке с файлом
3. Выполните команду: `python main.py`
4. При первом запуске приложение создаст файл `quotes_data.json` с базовой базой цитат

## 🧪 Примеры использования и тесты
1. **Базовая генерация**: запустите приложение → нажмите `Сгенерировать цитату` → убедитесь, что текст, автор и тема отобразились корректно
2. **Фильтрация**: выберите конкретного автора в выпадающем списке → нажмите кнопку генерации → проверяйте, что цитаты принадлежат только этому автору
3. **История**: сгенерируйте 5–10 цитат → прокрутите поле истории → убедитесь, что отображаются последние записи в хронологическом порядке
4. **Сохранение/загрузка**: нажмите `Сохранить данные` → закройте приложение → откройте заново → история и база цитат должны восстановиться автоматически
5. **Очистка**: нажмите `Очистить историю` → поле должно стать пустым, но базовые цитаты останутся доступными для генерации
